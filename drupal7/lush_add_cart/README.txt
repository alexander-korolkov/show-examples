INTRODUCTION
------------------------------------
This module provides "Add product to cart for ajax".
Need to output $classes variable on Each products
content(output $classes variable on the product template).
Module will add basket button by js.

Configuration url admin/config/lush/lush-cart
1 Set lush cart product select option.
Example:field name1/field name2 . like: commerce_price/field_net_weight_text.
Sure the filed name is in your product.
2 Select product content type.
